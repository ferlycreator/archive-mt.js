------

MapTalks地图引擎前端Javascript SDK库

开发环境建立步骤:
-----------------

1. 安装node.js
2. npm install
3. 运行 gulp


目录结构:
-----------------

src: 源代码

dist: 发布目录

build: 编译发布所需的资源文件

assets: css和images文件

test: 单元测试代码
