快速上手 MapTalks

该向导帮助您将快速掌握构建地图、标注点、线与面，并为它们设置标签，以及监听处理交互事件。

第一张定制地图

1、引入 MapTalks JavaScript 文件；

<link href="http://www.maptalks.org/maptalks/v2/css/maptalks.min.js" rel="stylesheet" type="text/css"/>

2、在页面中添加一个放置Map的Html Element，例如：DIV；

<div id="mapObj" style="width:100%; height:100%;"></div>

3、通过 new Z.Map(...)创建地图对象，并设置底图图层。

<!-- 第一步：MapTalks JavaScript 开发包 -->
<script type="text/javascript" src="http://www.maptalks.org/maptalks/v2/css/maptalks.min.js"></script>
<body scroll="no"  onload="initMap();">
    <!-- 第二步、设置放置地图的Html Element -->
    <div id="mapObj" style="width:100%;height:100%;float:right;margin-top:0px"></div>
</body>
<!-- 第三步、创建地图对象，并设置底图图层 -->
<script type="text/javascript" >
    var mapObj;
    function initMap(){
        //创建地图对象
        mapObj = new Z.Map('mapObj', //放置mapObj的div id
        {
            center:     new Z.Coordinate(121.489935,31.24432), //地图初始中心点坐标
            zoomLevel:  17, //地图初始缩放级别
            attributionControl: false, //版权信息控件
            zoomControl : false, // 地图缩放控件
            navControl: false, //地图导航控件
            scaleControl: false //地图比例尺控件
        });
        //定制地图底图(Tile Layer)
        var tileLayer = new Z.TileLayer('tile', //底图id
        {
            crs:'crs3857', //坐标系
            urlTemplate:'http://emap{s}.mapabc.com/mapabc/maptile?&x={x}&y={y}&z={z}',//底图服务url
            subdomains:[1,2,3]//地图服务url子域名变量,对应urlTemplate中占位符{s}的值
        });
        //设置底图图层
        mapObj.setBaseTileLayer(tileLayer);
    }
</script>

添加点

1、通过new Z.Marker(...)创建一个点；

2、将点添加到图层上。

//创建点的坐标对象
var coordinate = new Z.Coordinate(121.489935,31.24432);
//创建点对象
var marker = new Z.Marker(coordinate);
//将点添加到Layer
layer.addGeometry(marker);

标注图片

1、通过new Z.Marker(...)创建一个点；

2、构造图片参数，并设置到Marker；

3、将点添加到图层上。

//创建点的坐标对象
var coordinate = new Z.Coordinate(121.489935,31.24432);
//设置图片
var icon = {
    'marker-file': '/engine/images/marker.png',
    'marker-width': 22,
    'marker-height': 30,
    'dx': 0,
    'dy' :0
};
//创建点对象
var marker = new Z.Marker(coordinate, 'marker_id');
//设置点样式
marker.setSymbol(icon);
//将点添加到Layer
layer.addGeometry(marker);

标注矢量图形

1、通过new Z.Marker(...)创建一个点；

2、构造矢量图形参数，并设置到Marker；

3、将点添加到图层上。

//创建点的坐标对象
var coordinate = new Z.Coordinate(121.489935,31.24432);
//设置矢量配置
var icon = {
    'marker-type': 'triangle',
    'marker-opacity': 1,
    'marker-fill-opacity': 1,
    'marker-line-color': '#0000ff',
    'marker-line-width': 1,
    'marker-line-opacity': 1,
    'marker-placement': 'point',
    'marker-width': 30,
    'marker-height': 30,
    'marker-fill': '#ff0000',
    'dx': 0,
    'dy': 0
};
/**
* marker-type 说明：
* circle：圆形；triangle：三角形；cross：十字；diamond：菱形；square：正方形；x/X：X形；bar：柱状图
*/
//创建点对象
var marker = new Z.Marker(coordinate, 'marker_id');
//设置点样式
marker.setSymbol(icon);
//将点添加到Layer
layer.addGeometry(marker);

设置点标签

1、通过new Z.Marker(...)创建一个点；

2、构造标签参数，并设置到Marker；

3、将点添加到图层上。

//创建点的坐标对象
var coordinate = new Z.Coordinate(121.489935,31.24432);
//设置矢量配置
var icon = {
    'shield-type': 'tip',//label tip
    'shield-name': '测试标签: [marker_name]',
    'shield-opacity': 1,
    'shield-line-color': '#121212',
    'shield-line-width': 1,
    'shield-line-opacity': 1,
    'shield-fill': '#536457',
    'shield-fill-opacity': 1,
    'shield-file': '',
    'shield-face-name': 'Serif',
    'shield-unlock-image' : false,
    'shield-size': 12,
    'shield-text-fill': '#ff0000',
    'shield-placement': 'point', //point line vertex interior
    'shield-wrap-width': 100,
    'shield-wrap-before': false,
    'shield-wrap-character': '',
    'shield-character-spacing': 0,
    'shield-line-spacing': 8,
    'shield-text-dx': 0,
    'shield-text-dy': 0,
    'shield-dx': 0,
    'shield-dy': 0,
    'shield-text-opacity': 1,
    'shield-horizontal-alignment': 'middle',//left middle right
    'shield-vertical-alignment': 'top',//top middle bottom
    'shield-justify-alignment': 'left'//left center right
};
/**
* 说明：
* 1、shield-type tip：气泡形；label：矩形标签；
* 2、在shield-type 为tip的情形下，设置shield-horizontal-alignment 与 shield-vertical-alignment
*    气泡的尖端会根据位置调整。
*/
//创建点对象
var marker = new Z.Marker(coordinate, 'marker_id');
marker.setProperties({
    'marker_name':'上海大厦',
    'marker_addr': '北苏州路20号'
});
//设置点样式
marker.setSymbol(icon);
//将点添加到Layer
layer.addGeometry(marker);
