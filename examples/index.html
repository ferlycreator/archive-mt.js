添加点

通过new Z.Marker(...)创建一个点，并添加到图层上。

//创建点的坐标对象
var coordinate = new Z.Coordinate(121.489935,31.24432);
//创建点对象
var marker = new Z.Marker(coordinate);
//将点添加到Layer
layer.addGeometry(marker);

标注图片

1、通过new Z.Marker(...)创建一个点；

2、构造图片参数，并设置到Marker；

3、将点添加到图层上。

//创建点的坐标对象
var coordinate = new Z.Coordinate(121.489935,31.24432);
//设置图片
var icon = {
    'marker-file': '/engine/images/marker.png',
    'marker-width': 22,
    'marker-height': 30,
    'dx': 0,
    'dy' :0
};
//创建点对象
var marker = new Z.Marker(coordinate,'marker_id');
//设置点样式
marker.setSymbol(icon);
//将点添加到Layer
layer.addGeometry(marker);

标注矢量图形

1、通过new Z.Marker(...)创建一个点；

2、构造矢量图形参数，并设置到Marker；

3、将点添加到图层上。

//创建点的坐标对象
var coordinate = new Z.Coordinate(121.489935,31.24432);
//设置矢量配置
var icon = {
    'marker-type': 'triangle',
    'marker-opacity': 1,
    'marker-fill-opacity': 1,
    'marker-line-color': '#0000ff',
    'marker-line-width': 1,
    'marker-line-opacity': 1,
    'marker-placement': 'point',
    'marker-width': 30,
    'marker-height': 30,
    'marker-fill': '#ff0000',
    'dx': 0,
    'dy': 0
};
/**
* marker-type 说明：
* circle：圆形；triangle：三角形；cross：十字；diamond：菱形；square：正方形；x/X：X形；bar：柱状图
*/
//创建点对象
var marker = new Z.Marker(coordinate, {symbol:icon});
//设置点样式
marker.setSymbol(icon);
//将点添加到Layer
layer.addGeometry(marker);
